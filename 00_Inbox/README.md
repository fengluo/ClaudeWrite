# 📥 Inbox - 收件箱

这是你的快速捕获区，用于临时存放新想法、灵感和待处理的内容。

## 使用方法

### 快速捕获
使用 `/quick-capture` 命令快速记录想法:
```bash
/quick-capture
```

### 定期整理
建议每天或每周使用 `/inbox-processor` 清空收件箱:
```bash
/inbox-processor
```

## 最佳实践

1. **零摩擦捕获**: 不要担心格式，先记录下来
2. **不要久留**: 收件箱不是永久存储，应定期整理
3. **添加标签**: 捕获时简单标记 (#idea, #todo, #question)
4. **快速分类**: 整理时移动到对应的项目/领域/资源文件夹

## 文件命名约定

使用日期前缀便于排序:
```
YYYY-MM-DD-brief-title.md
```

例如:
- `2024-01-10-react-hooks-idea.md`
- `2024-01-10-meeting-notes.md`
- `2024-01-10-question-about-api.md`
