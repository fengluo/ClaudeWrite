# Draft Content - 初稿生成

根据大纲或笔记快速生成文章初稿。

## 工作流程

1. **分析输入**
   - 读取当前文件中的大纲或笔记
   - 这里的"输入"通常是结构化的 bullet points

2. **生成策略**
   - 保持原有的结构和标题层级
   - 将 bullet points 扩展为完整的段落
   - 使用自然的连接词
   - 遇到不确定的细节，使用 `[需要补充: ...]` 标记

3. **写作风格**
   - 清晰、直接、专业
   - 避免过度的修饰词
   - 保持逻辑连贯性

## 示例

```
User: /draft-content
(当前文件包含一个关于"如何学习编程"的大纲)

AI: 我将基于你的大纲生成初稿。

对于初学者来说，选择正确的入门语言至关重要。

Python 是目前最推荐的选择。它的语法简洁清晰，接近自然语言，让初学者能专注于编程逻辑而不是复杂的语法规则。此外，Python 在数据科学、人工智能和 Web 开发等领域都有广泛应用。

JavaScript 是另一个不错的选择，特别是如果你对网页开发感兴趣。它是 Web 前端的通用语言，学习它能让你快速看到视觉上的反馈。
```

## 行为准则

### 必须做 ✓
- ✓ 严格遵循用户提供的大纲结构
- ✓ 标记出数据缺失或逻辑跳跃的地方
- ✓ 保持语气的一致性

### 禁止做 ✗
- ✗ 不要随意添加大纲中没有的观点
- ✗ 不要生成虚假的数据或案例
- ✗ 不要使用过于浮夸的营销语言
