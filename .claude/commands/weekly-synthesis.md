# Weekly Synthesis - 每周综合

每周进行一次深度回顾，整合本周进展，提炼洞察，并为下周制定战略计划。

## 工作流程

### 1. 扫描本周活动

使用 Bash 工具查找过去 7 天修改的文件：

```bash
# 查找过去 7 天创建的文件
find . -type f -name "*.md" -newermt "7 days ago" ! -path "./node_modules/*" ! -path "./.git/*"

# 查找过去 7 天修改的文件
find . -type f -name "*.md" -newermt "7 days ago" ! -path "./node_modules/*" ! -path "./.git/*"
```

统计并分类：
- 本周新增笔记数
- 本周更新笔记数
- 活跃项目分布
- 重点关注领域

### 2. 生成周度摘要

格式化展示本周活动概览：

```
📅 本周活动统计 (YYYY-MM-DD 至 YYYY-MM-DD)

## 核心数据
- 新增笔记: X 篇
- 更新笔记: Y 篇
- 活跃天数: Z/7 天

## 重点推进项目
1. [[Project-A]]
   - 关键进展: ...
   - 修改文件数: ...

2. [[Project-B]]
   - 关键进展: ...
   - 修改文件数: ...

## 知识图谱生长
02_Areas/
- [[Area-1]]: 新增了关于 ... 的知识
- [[Area-2]]: 完善了 ... 的结构

## 外部输入
00_Inbox/ & 03_Resources/
- 新收集资源: X 个
- 待处理收件箱项: Y 个
```

### 3. 综合回顾与洞察

引导用户进行深度反思：

```
## 深度反思时刻

请花些时间回顾这一周的整体表现：

### 1. 本周高光时刻 (Big Wins)
这一周最令你满意的成就是什么？完成了哪些里程碑？
(关注结果和影响力)

> [等待用户输入]

### 2. 阻碍与瓶颈
本周遇到了哪些反复出现的问题或阻碍？
(不仅仅是具体问题，更关注流程或习惯上的瓶颈)

> [等待用户输入]

### 3. 关键洞察
本周学到的最重要的经验或对系统/工作的认知是什么？
(这些洞察值得被提取到永久笔记中吗？)

> [等待用户输入]

### 4. 遗留事务
哪些本该完成但未完成的任务？原因是什么？
(是计划不合理，还是执行力问题，或者是优先级调整？)

> [等待用户输入]
```

### 4. 下周战略规划

制定下周计划：

```
## 下周规划

基于本周的回顾，让我们规划下周：

### 1. 下周主要目标 (Top 3)
下周必须要完成的 3 件事是什么？

> [等待用户输入]

### 2. 重点关注项目
下周将把精力集中在哪些项目上？

> [等待用户输入]

### 3. 改进尝试
基于本周的"阻碍与瓶颈"，下周打算尝试什么小的改进？

> [等待用户输入]
```

### 5. 创建周度回顾记录

基于用户回答，使用 Write 工具创建：

**文件路径**: `06_Meta/Reviews/weekly/YYYY-Www.md` (例如: 2024-W02.md)

**内容格式**：
```markdown
---
title: Weekly Review - YYYY-Www
date: YYYY-MM-DD
tags: [weekly-review, reflection, planning]
week_start: YYYY-MM-DD
week_end: YYYY-MM-DD

## 下周规划

### 主要目标
```

### 6. 完成提示与后续行动

创建成功后：

```
✅ 周度回顾已保存到 06_Meta/Reviews/weekly/YYYY-Www.md

下周重点:
1. [目标1]
2. [目标2]
3. [目标3]

建议行动:
- 将"改进尝试"添加到下周的任务清单中
- 检查"遗留事务"，决定是安排到下周、推迟还是删除
- 如果有重要的"关键洞察"，建议现在就运行 /quick-capture 将其转化为永久笔记

祝你下周工作顺利！🚀
```

## 行为准则

### 必须做 ✓
- ✓ 关注长期趋势而非短期波动
- ✓ 鼓励用户庆祝胜利，增强动力
- ✓ 帮助用户识别系统性问题而非单纯责怪自己
- ✓ 确保下周计划是现实可行的

### 禁止做 ✗
- ✗ 不要让周回顾变成简单的流水账
- ✗ 不要忽略未完成的任务，要分析原因
- ✗ 不要制定过于庞大或模糊的下周目标

## 最佳实践
- **时间**: 建议在周五下午或周日晚上进行
- **环境**: 找一个安静的时间，不受打扰
- **心态**: 保持客观，像通过上帝视角审视自己的一周
