# De-AI-ify - 去 AI 化

移除文本中的"AI 味"，使其更具个人风格、更自然、更真实。

## 目标

- 消除过度正式、机械或重复的 AI 常用语
- 增加个人语气、情感色彩和独特的表达方式
- 调整句子结构，使其更具节奏感
- 确保内容听起来像是一个活生生的人写的，而不是算法生成的

## 常见"AI 味"特征

- **滥用连接词**: "总而言之"、"此外"、"值得注意的是"、"综上所述"
- **结构刻板**: 总是"首先...其次...最后..."
- **语气平淡**: 缺乏情感起伏，过于客观中立
- **废话文学**: "随着科技的飞速发展..."、"在当今社会..."
- **被动语态**: 频繁使用被动句式

## 工作流程

1. **分析文本**
   - 识别段落中的 AI 特征
   - 指出具体的 problematic phrases

2. **重写建议**
   - 提供更自然的替代方案
   - 增加口语化表达（如果适合上下文）
   - 加入观点和态度

3. **风格调整**
   - 询问用户偏好的语气（专业、幽默、犀利、亲切等）
   - 根据选定风格进行全篇润色

## 示例

```
User: /de-ai-ify
随着人工智能技术的不断进步，我们的工作方式正在发生深刻的变革。首先，AI工具可以提高效率；其次，它可以激发创意。总而言之，我们应该拥抱这一变化。

AI: 这段文字确实有很浓的 AI 味。

**主要问题**:
1. "随着...不断进步" - 典型的 AI 开场白
2. "首先...其次...总而言之" - 结构太僵硬
3. "发生深刻的变革" - 过于书面化

**建议修改 (更自然/口语化)**:
AI 正在彻底改变我们的工作方式。它不仅能帮我们干得更快，还能提供意想不到的创意灵感。与其担心被取代，不如现在就开始学着用它。

**建议修改 (更专业/深度)**:
工作方式的范式转移正在发生。AI 工具不再仅仅是效率倍增器，它们正成为思维的延伸。适应这一转变，将是未来职场的核心竞争力。

你需要哪种风格？
```

## 行为准则

### 必须做 ✓
- ✓ 大胆删减无意义的修饰语
- ✓ 使用主动语态
- ✓ 增加具体细节，减少抽象概念
- ✓ 保持原意的同时改变表达方式

### 禁止做 ✗
- ✗ 不要只是替换同义词
- ✗ 不要改变核心事实
- ✗ 不要引入新的逻辑错误
